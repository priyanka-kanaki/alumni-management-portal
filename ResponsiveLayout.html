About the Project
( Overview )

The project is an AlumniSphere: A Smart Portal for Alumni wherein the passed out Alumni's can view their graduation progress information like batch, projects undertaken, their details, etc. It is a complete database where by logging in you can access your full details of graduation. The project contains different logins : College, Student, Alumni. Here the alumni can edit there information like name of company currently working in. Also they can get latest updates regarding the Alumni Meet or the events going on in the college. They can also have chat with the admin for if any queries. They can access some information of the fellow alumnis. They can also send messages to their batchmates which will land in their mail.
